## 核心原则调整

- **P0**：项目的核心差异点，没有就不是 Miniese's Blog
- **P1**：重要但可以晚一两个版本
- **P2**：锦上添花

**所有 AI 辅助创作能力（审查、关联、翻译、词条生成）都是 P0。**

## MVP 总结（v0.2）

| 模块 | MVP 包含 |
|------|----------|
| 文章 | 发布、草稿、渲染、列表页、阅读页 |
| 知识库 | 手动词条 + AI 发现与生成 + 自动链接 + hover 预览 |
| AI | 文章对话、changelog、关联检测、审查报告、手动翻译、词条生成 |
| 仪表盘 | 文章/词条管理、词条提议审批、审查报告查看、翻译触发 |
| 账户 | 仅博主 |
| 前端 | 所有 P0 交互 |
